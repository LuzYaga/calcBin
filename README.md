# Conversor de número decimal para binário

Atividade feita na faculdade, conversor básico de número decimal para número binário, utilizando o conceito de pilhas, aprendido na matéria de Estrutura de Dados.


## 🚀 Como executar

```bash
# Clone o projeto
git clone https://github.com/LuzYaga/calcBin.git

# Entre na pasta
cd src

# Compile
javac Pilha_INT.java

# Execute
java Pilha_INT
